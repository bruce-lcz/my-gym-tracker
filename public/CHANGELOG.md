# 版本紀錄

## v1.3.0
**發布日期：** 2025-12-18
**類型：** feature
**標題：** 統計儀表板與範例資料功能

### 更新內容
- **新增統計儀表板 (Dashboard)** - 可視化訓練數據，包含部位分佈圖與常練動作排行
- **新增範例資料 (Mock Data)** - 支援一鍵載入 Demo 資料，方便預覽圖表功能

## v1.2.0
**發布日期：** 2025-12-18
**類型：** feature
**標題：** 手機版 UI 優化與時區修正

### 更新內容
- **手機版優化** - 修正跑版問題，新增滑入式側邊欄
- **時區修正** - 確保訓練日期正確顯示本地時間
- 新增手機版浮動選單按鈕，側邊欄改為滑入式
- 新增 Vite server 設定，支援手機瀏覽 localhost
- 版本紀錄改用 CHANGELOG.md 管理，更易於維護

---

## v1.1.0
**發布日期：** 2025-12-17
**類型：** feature
**標題：** 圖標系統升級與版本資訊功能

### 更新內容
- 整合 Lucide React 圖標庫，提供更美觀的視覺體驗
- 新增 Tab 導航系統，分離新增訓練、訓練紀錄和版本資訊
- 優化所有按鈕設計，加入對應的圖標提升使用體驗
- 新增勵志語錄系統，每次開啟隨機顯示激勵文字
- 新增版本資訊頁面，展示應用功能與更新歷史

## v1.0.0
**發布日期：** 2025-12-17
**類型：** feature
**標題：** 健身追蹤器首次發布

### 更新內容
- 基本訓練記錄功能 - 記錄動作、組數、重量、RPE
- 訓練歷史查詢 - 查看所有歷史訓練記錄
- Google Sheets 整合 - 雲端同步訓練數據
- 深色/淺色主題切換 - 根據喜好自訂介面
- 自訂動作功能 - 可新增個人化的訓練動作
- 響應式設計 - 完美支援手機、平板、桌面裝置
