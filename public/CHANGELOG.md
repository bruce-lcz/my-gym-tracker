# 版本紀錄


## v1.9.0
**發布日期：** 2026-01-07
**類型：** feature
**標題：** 自訂動作同步與訓練紀錄 UI 改進

### 更新內容
- **自訂動作同步至 Google Sheets** - 新增動作時自動同步至後端，建立 Exercises 分頁，提供重複檢測與備援機制
- **訓練紀錄日期分組** - 依日期自動分組顯示，新增固定式日期標題與日曆圖示
- **卡片式佈局** - 採用現代卡片設計取代傳統表格，資訊層次更清晰
- **響應式優化** - 手機版單欄佈局，觸控友善設計
- **訓練組數順序修正** - 修正統計儀表板與今日訓練的組數顯示順序
